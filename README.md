# gxDNN #

Run Neural networks on NationalChip NPU processor.

## Installation ##

### 安装gxDNN工具链 ###

> pip install -i https://testpypi.python.org/pypi npu_compiler

### 更新gxDNN工具链 ###

> pip install --upgrade -i https://testpypi.python.org/pypi npu_compiler

### 查看工具链版本 ###

安装或更新完成后，可以使用如下命令查看当前工具链的版本号。

> gxnpuc --version
